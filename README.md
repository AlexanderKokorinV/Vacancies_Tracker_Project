# Vacancies_Tracker_Project

## Описание

Проект Vacancies_Tracker_Project представляет собой серверную часть приложения для сбора данных о компаниях и их вакансиях с сайта hh.ru, сохранения их в базу данных PostgreSQL и последующего анализа.

После запуска вам будут доступны следующие действия:
 - Вывод списка компаний и количества их вакансий.
 - Просмотр всех вакансий с зарплатами и ссылками.
 - Расчет средней зарплаты по всей базе.
 - Поиск вакансий с зарплатой выше средней.
 - Поиск вакансий по ключевому слову в названии.

Проект состоит из следующих модулей:
 1. Модуль src/hh_api.py (класс HHParser) отвечает за взаимодействие с внешним API (hh.ru)
    Выполняет запросы к публичному API hh.ru. Получает первичные данные о компаниях (get_employers) и детальные списки вакансий (get_vacancies).
    Используется библиотека requests.
2. Модуль src/utils.py содержит следующие функции:
    create_database: Автоматизирует создание БД и таблиц employers и vacancies в PostgreSQL.
    save_to_db: Трансформирует данные из JSON-ответов API в реляционный вид и сохраняет их в таблицы.
3. Модуль src/db_manager.py (класс DBManager) отвечает за подключение к БД PostgreSQL и анализ данных. Содержит набор SQL-методов для выборки данных: подсчет количества вакансий, расчет средней зарплаты, фильтрация по ключевым словам и уровню дохода.
    Используется библиотека psycopg2 для работы с БД
4. Модуль main.py - точка входа и интерфейс. Инициализирует все компоненты, координирует процесс парсинга и предоставляет пользователю интерактивное консольное меню для работы с данными.


В проекте применены следующие инструменты и технологии:
- **Python 3.10+**
- **PostgreSQL** (СУБД)
- **psycopg2** (драйвер для БД)
- **requests** (работа с API)
- **pytest** / **coverage** (тестирование)
- **mypy** (проверка типизации)

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/AlexanderKokorinV/Vacancies_Tracker_Project.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Запуск программы

```
python main.py
```

## Тестирование

Проект использует `pytest` для написания и выполнения тестов. 
Тесты используют Mock и patch.
Процент покрытия тестами (coverage rate) расположен в файле index.html

Чтобы запустить тесты, следуйте инструкциям ниже:
1. Убедитесь, что `pytest` установлен. Если нет, установите его командой:
```
pip install pytest
```
2. В корневой директории проекта выполните команду для запуска всех тестов:
```
pytest
```
3. Если необходимо, вы можете указать конкретный файл теста:
```
pytest tests/test_utils.py
```
Это позволит убедиться, что проект работает корректно и все функции проходят проверку.

4. Для проверки покрытия тестами воспользуйтесь командой:
```
pytest --cov=src --cov-report=html
```
